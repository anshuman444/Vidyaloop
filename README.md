# VidyaLoop PDA — Final Master Build

## Behavioral Intelligence Assessment Platform

VidyaLoop is a student-centric behavioral intelligence platform that analyzes cognitive and emotional patterns to provide deep psychological insights.

### Tech Stack
- **Frontend**: React.js
- **Styling**: Tailwind CSS v4 (Glassmorphism, Modern UI)
- **Animations**: Framer Motion
- **Data Visualization**: Recharts (Radar charts, gauge charts)
- **Icons**: Lucide React

### Core Logic Engine (9 Layers)
1. **Response Processing**: Handles reverse scoring and normalization.
2. **Dimension Scoring**: Calculates raw scores and percentiles for 5 key dimensions.
3. **Consistency Engine**: Validates response reliability.
4. **Latent Trait Engine**: Infers secondary traits like Resilience and Leadership.
5. **Pattern Engine**: Detects cross-dimension relationships.
6. **Risk Engine**: Identifies behavioral tendencies like burnout risk.
7. **Archetype Engine**: Assigns one of several psychological archetypes.
8. **Explainability Engine**: Provides the "Why" behind the patterns.
9. **Narrative Engine**: Synthesizes a personalized, believable report.

### Development
- `npm install`: Install dependencies
- `npm run dev`: Start the intelligence platform locally

*Note: This platform runs entirely client-side. No data ever leaves the browser.*
